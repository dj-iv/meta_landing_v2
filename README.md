# Meta Landing v2

Curiosity-first landing page for UCtel Office Signal Solutions.

## What's Different

This version shifts from direct sales to education-first:

| Before | After |
|--------|-------|
| "Eliminate Mobile Dead Zones" | "There's a reason your corner office gets no bars..." |
| Form asking for work email + office description | Email-only form for PDF checklist |
| Immediate "book assessment" CTA | Educational flow: problem → explanation → solution |

## Key Features

- **Curiosity-aligned headline** matches the winning Meta ad (4.6% CTR)
- **Email-only capture** for "Office Manager's Signal Checklist" PDF
- **Educational flow** that educates before asking for commitment
- **Secondary CTA** for those ready to book a call now
- **Meta Pixel** and **Google Analytics** integrated
- **Real client logos** from original landing page
